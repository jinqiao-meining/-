## 二、逐题简析

1. 机器语言直接面向机器，不属于高级语言。
2. Python 是动态类型语言，不是静态语言。
3. `_3_` 符合变量命名规则；关键字和数字开头都不行。
4. `gender == "male"` 是比较表达式，不是赋值语句。
5. Python 没有 `avg()` 这个内置函数，常见的是 `sum()`、`max()`、`min()`。
6. `1` 是整数，所以 `type(1)` 是 `<class 'int'>`。
7. `True` 是布尔值，所以类型是 `<class 'bool'>`。
8. `"1"` 带引号，是字符串类型。
9. `[1]` 使用方括号，表示列表。
10. `(1)` 只是加了括号的整数，不是元组。
11. `(1,)` 末尾有逗号，才表示单元素元组。
12. `{1}` 表示集合，不是字典。
13. `{1:1, 2:2}` 是键值对结构，所以是字典。
14. f-string 会把变量直接替换到字符串中，输出为 `Our Python tearcher is Li Xuefeng.`。
15. `%s` 只替换了 `course`，`{name}` 不会被自动替换。
16. 依次计算得：104、98、303、33、33、2、1030301。
17. 单利：100 × (1 + 0.04 × 30) = 220；复利：100 × 1.04^30 ≈ 324.34。
18. 原始字符串 `r""` 不会把 `\n` 当成换行，所以按原样输出。
19. 普通字符串里 `\n` 会换行，`\\` 表示一个反斜杠。
20. `%` 格式化会把 `%d`、`%i`、`%s` 依次替换成对应值。
21. `%3.1f%%` 中前一部分控制保留 1 位小数，后一个 `%` 用来输出百分号。
22. Python 单行注释符号是 `#`。
23. 多行说明常用一对三引号表示。
24. 程序三大基本控制结构是顺序、分支、循环，没有“逐步结构”。
25. `if ... else ...` 要注意冒号和缩进。
26. `50.46` 为真，且 `4.1f` 会四舍五入保留 1 位小数，得到 `50.5`。
27. `range(4)` 生成 0 到 3，转成列表是 `[0, 1, 2, 3]`。
28. `range(2)` 生成 0、1，转成元组是 `(0, 1)`。
29. `range(1, 2)` 只有一个元素 1，单元素元组写作 `(1,)`。
30. 字符串、列表、元组的长度都为 5；集合有 3 个元素；字典有 3 个键。
31. `range(4, 100, 2)` 是从 4 到 98，共 48 个数，和为 2448。
32. 这题是自我评价题；若按完成作业的角度填写，可选 `C`。
33. `range(100)` 会循环 100 次，`end=''` 可以让星号连续输出在同一行。

## 三、知识补充

### 1. 高级语言、低级语言

- 高级语言更接近人类思维，编写和维护更方便，比如 Python、Java、C。
- 机器语言直接由 0 和 1 组成，执行快但可读性差。

### 2. 静态语言与动态语言

- 静态语言：变量类型通常在编译阶段就更明确，如 Java、C、C++。
- 动态语言：变量类型可以在运行时变化，Python 是典型代表。

### 3. Python 变量命名规则

- 只能由字母、数字、下划线组成。
- 不能以数字开头。
- 不能使用关键字，例如 `if`、`for`、`in`。
- 推荐见名知意，例如 `student_name`、`total_score`。

### 4. 常见数据类型识别

- `1` 是 `int`
- `1.0` 是 `float`
- `"1"` 是 `str`
- `True` 是 `bool`
- `[1]` 是 `list`
- `(1,)` 是 `tuple`
- `{1}` 是 `set`
- `{1: 1}` 是 `dict`

### 5. 三种常见字符串格式化方式

- f-string：`f"姓名是{name}"`
- 百分号格式化：`"姓名是%s" % name`
- `format()`：`"姓名是{}".format(name)`

其中，f-string 最直观，也最推荐初学者掌握。

### 6. `range()` 的核心规律

- `range(4)` -> 0, 1, 2, 3
- `range(2, 6)` -> 2, 3, 4, 5
- `range(1, 10, 2)` -> 1, 3, 5, 7, 9

记忆口诀：包左不包右，步长默认 1。

### 7. `len()`、`sum()` 与循环

- `len()` 用来统计元素个数。
- `sum()` 用来求数值总和。
- `for` 循环常用来做“计数、遍历、求和”这三类题。

### 8. 初学者最容易错的点

- 把 `=` 和 `==` 混淆。
- 把 `(1)` 误认为元组，实际上 `(1,)` 才是元组。
- 忘记缩进。
- 忘记 `range()` 右边取不到。
- 误把字符串 `"1"` 当成整数 `1`。

## 四、速记小结

Python 第一章最重要的不是死记题目，而是建立 5 个基本感觉：

1. 看得懂变量和类型。
2. 分得清赋值和比较。
3. 熟悉字符串格式化。
4. 能判断分支和循环的基本语法。
5. 会用 `range()`、`len()`、`sum()` 解决基础问题。

如果把这 5 点掌握住，后面学习列表、函数、文件、面向对象都会轻松很多。
