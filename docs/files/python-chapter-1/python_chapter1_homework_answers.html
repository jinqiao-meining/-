复习提醒

变量命名不能数字开头，也不能和关键字重名。

range() 记住“包左不包右”，(1,) 才是单元素元组。

逐题简析

  1. 机器语言不属于高级语言。
  2. Python 是动态语言，不是静态语言。
  3. _3_ 符合变量命名规则。
  4. gender == "male" 是比较，不是赋值。
  5. Python 没有内置函数 avg()。
  6. type(1) 是 <class 'int'>。
  7. True 的类型是 bool。
  8. "1" 是字符串 str。
  9. [1] 是列表 list。
  10. (1) 只是整数加括号，不是元组。
  11. (1,) 才是元组。
  12. {1} 是集合 set。
  13. {1:1, 2:2} 是字典 dict。
  14. f-string 会直接替换变量，所以选 A。
  15. 只替换了 %s，{name} 会原样输出，所以选 C。
  16. 依次为 104、98、303、33、33、2、1030301。
  17. 单利 220，复利约 324.34。
  18. 原始字符串不会把 \n 当换行，选 B。
  19. 普通字符串里 \n 会换行，\\ 表示一个反斜杠，选 B。
  20. 百分号格式化后整句会被正常替换，选 A。
  21. %3.1f%% 会输出保留 1 位小数的百分数，选 A。
  22. 单行注释是 #。
  23. 多行说明常见写法是一对三引号。
  24. 程序三大基本结构是顺序、分支、循环，没有“逐步结构”。
  25. 正确语法要有冒号，也要有缩进，所以选 B。
  26. 50.46 保留一位小数得到 50.5。
  27. range(4) 转列表是 [0, 1, 2, 3]。
  28. range(2) 转元组是 (0, 1)。
  29. range(1, 2) 只有一个元素，结果是 (1, )。
  30. 五个值分别是 5、5、5、3、3。
  31. range(4, 100, 2) 共 48 个数，和为 2448。
  32. 这是自评题；按完成作业填写，通常选 C。
  33. range(100) 循环 100 次，end='' 保证在同一行连续输出。

知识补充

1. 高级语言和机器语言

  • 高级语言更接近人的思维，代码更容易写和维护，比如 Python、Java、C。
  • 机器语言直接面向硬件执行，可读性很差。

2. 静态语言与动态语言

  • 静态语言：变量类型通常在编译阶段更明确，如 Java、C、C++。
  • 动态语言：变量类型可以在运行中变化，Python 是典型代表。

3. Python 变量命名规则

  • 只能由字母、数字、下划线组成。
  • 不能以数字开头。
  • 不能使用关键字，如 for、if、in。
  • 推荐使用见名知意的命名，如 student_name。

4. 常见数据类型速认

  • 1 是 int
  • 1.0 是 float
  • "1" 是 str
  • True 是 bool
  • [1] 是 list
  • (1,) 是 tuple
  • {1} 是 set
  • {1: 1} 是 dict

5. 三种字符串格式化方式

  • f-string：f"姓名是{name}"
  • 百分号格式化："姓名是%s" % name
  • format()："姓名是{}".format(name)

6. range() 核心规律

  • range(4) -> 0, 1, 2, 3
  • range(2, 6) -> 2, 3, 4, 5
  • range(1, 10, 2) -> 1, 3, 5, 7, 9

记忆口诀：包左不包右，步长默认 1。

7. len()、sum() 和 for 循环

  • len() 用来统计元素个数。
  • sum() 用来求数值总和。
  • for 循环最常见的三类基础题：计数、遍历、求和。

8. 初学者最容易错的地方

  • 把 = 和 == 混淆。
  • 把 (1) 误认成元组。
  • 忘记缩进。
  • 忘记 range() 右边取不到。
  • 把字符串 "1" 当成整数 1。

5 个速记要点

1看懂变量和类型。
2分清赋值和比较。
3掌握字符串格式化。
4看懂分支和循环语法。
5会用 range()、len()、sum()。

这 5 点掌握住，后面学习列表、函数、文件和面向对象会轻松很多。
