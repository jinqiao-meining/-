这个仓库怎么用

这是一个静态站点仓库。你把内容文件放进仓库并推送到公开 GitHub 仓库后，站点会自动更新。默认最省钱、最稳定的方案就是 GitHub Pages。

后续发布流程

  1. 准备新 PDF、HTML 或资料文件。
  2. 运行上传脚本，把文件加入站点目录。
  3. 提交并推送到 GitHub。
  4. 公开链接自动更新，所有人都能访问。
